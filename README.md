# Ven's Portfolio

An interactive, sticker-wall personal portfolio for **Venkatesh (Ven)** — Sydney-based, finance · accounting · commercial.

Click and drag the stickers — each one does something (plays a song, links out, fires a popup). The centre card cycles through who I am, and there's a live stock-portfolio page.

## Pages
- `index.html` — the interactive sticker wall + intro card
- `stocks.html` / `stocks.js` — live personal stock portfolio (returns only, prices via Finnhub)
- `learn.html` — "Learn with Ven" (coming soon)

## Built with
Plain HTML/CSS/JS — no build step. Fonts via Google Fonts; live quotes via Finnhub; global click counter via counterapi.dev; song via the official YouTube IFrame player.
